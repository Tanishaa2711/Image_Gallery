# Image Gallery

A responsive and interactive Image Gallery web application built using HTML, CSS, and JavaScript. The project allows users to browse and view a collection of images in an organized and visually appealing layout.

## Features
- Responsive design for desktop and mobile devices
- Clean and modern user interface
- Image preview and display functionality
- Easy navigation through images
- Lightweight and user-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project was developed to practice frontend web development concepts, including responsive layouts, styling, and DOM manipulation using JavaScript.
